## 下载

### Docker

```
docker pull apachecn0/flygon-javaweb-zh
docker run -tid -p <port>:80 apachecn0/flygon-javaweb-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-javaweb-zh
flygon-javaweb-zh <port>
# 访问 http://localhost:{port} 查看文档
```
